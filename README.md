<b>Как получить дробную часть числа?</b>

Напишите функцию <b>getDecimal(num)</b>, которая возвращает десятичную часть числа:

alert( getDecimal(12.345) ); // 0.345 <br>
alert( getDecimal(1.2) ); // 0.2 <br>
alert( getDecimal(-1.2) ); // 0.2
